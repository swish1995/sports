# 생활스포츠지도사 2급 시험 준비 사이트

생활스포츠지도사 2급 자격증 필기/구술시험을 준비하기 위한 로컬 전용 학습 도구입니다.

## 주요 기능

- **필기시험**: 7과목 중 5과목 선택, 과목당 20문항 랜덤 출제, 100분 타이머
- **구술시험**: 보디빌딩/육상 종목 선택, 4문항 출제, 자기채점
- **자동 채점**: 과목별 40% + 전체 60% 합격 판정
- **오답노트**: 틀린 문제만 모아서 재시험
- **문제 편집**: 지문/보기/정답 수정, 코멘트(힌트) 추가
- **문제 임포트**: JSON/PDF 파일로 문제 일괄 등록
- **기출 정보**: 연도, 유형(A/B형), 문항번호 표시 (ℹ 아이콘 툴팁)
- **코멘트 힌트**: 시험 중 💡 아이콘으로 개인 메모 확인

## 시험 정보

| 항목 | 내용 |
|---|---|
| 필기시험 | 7과목 중 5과목 선택 (필수과목 없음) |
| 과목 | 스포츠사회학, 스포츠교육학, 스포츠심리학, 한국체육사, 운동생리학, 운동역학, 스포츠윤리 |
| 문항 수 | 과목당 20문항, 총 100문항 |
| 시험 시간 | 100분 |
| 문제 형식 | 4지선다 객관식 |
| 합격 기준 | 과목별 40% 이상 + 전체 평균 60% 이상 |
| 구술시험 | 4문항, 70% 이상 합격 |

## 빠른 시작

```bash
docker compose up -d
# http://localhost:8080 접속
```

자세한 배포 방법은 [docs/DEPLOY.md](docs/DEPLOY.md)를 참조하세요.

## 기술 스택

- **Backend**: Python Flask + SQLite
- **Frontend**: Vanilla HTML/CSS/JS (Jinja2 템플릿)
- **Container**: Docker (단일 컨테이너)

아키텍처 상세는 [docs/ARCHITECTURE.md](docs/ARCHITECTURE.md)를 참조하세요.

## 현재 문제 현황

2018~2025년 기출문제 약 860문제 수록 (PDF 파싱 기반).
파싱 품질이 완벽하지 않으므로 `/edit` 페이지에서 수동 교정 필요.

## 문제 추가 방법

### JSON 파일
```json
[
  {
    "subject": "스포츠윤리",
    "type": "multiple_choice",
    "question": "문제 텍스트",
    "options": ["보기1", "보기2", "보기3", "보기4"],
    "answer": "A",
    "exam_year": "2025",
    "exam_type": "A",
    "question_number": 1
  }
]
```
`/manage` 페이지에서 업로드하거나 `questions/` 디렉토리에 넣고 재시작.

### 수동 입력
`/edit` 또는 `/manage` 페이지에서 직접 추가.
